# AI StudyStation

### AI StudyStation is a Modern Academic Platform Landing Page Built with Next.js and Tailwind CSS

AI StudyStation revolutionizes the way students and educators manage their academic workflows by combining AI-powered note generation with seamless printing services.

<br/>

## ðŸ“¦ Installation

1. Clone the repository:

```bash
git clone https://github.com/yourusername/ai-studystation.git
cd ai-studystation
```

2. Install dependencies:

```bash
npm install
```

3. Add custom fonts to `public/fonts/`:

   - `JosefinSans-Regular.ttf`
   - `Nofret Light Regular.otf`

4. Run the development server:

```bash
npm run dev
```

5. Open [http://localhost:3000](http://localhost:3000) in your browser
